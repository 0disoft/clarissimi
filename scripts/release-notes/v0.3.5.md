# Clarissimi {{VERSION}}

Clarissimi {{VERSION}} makes provider-result failures easier to diagnose and adds maintainer-owned
model evaluation without widening the Action's trust boundary.

## Included

- a bounded provider-quality failure table in the GitHub Actions Step Summary
- failure summaries that exclude raw model output, provider messages, prompts, and credentials
- a scrubbed observed-failure corpus intake contract for real provider incidents
- a credential-free multi-model evaluation check for corpus and configuration drift
- an explicit manual-only live multi-model evaluation path for maintainer-owned credentials and
  provider cost
- the regenerated committed GitHub Action runtime bundle

## Unchanged boundaries

- no Action input, output, permission, or workflow-trigger changes
- no automatic maintainer approval
- no raw provider content in Action summaries or committed failure fixtures
- no npm or workspace-package publication
