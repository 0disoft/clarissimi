# Clarissimi {{VERSION}}

Clarissimi {{VERSION}} adds the opt-in, SHA-bound pre-merge review gate to the repository-owned
GitHub Action while preserving the existing recognition workflows.

## Added

- `mode: gate` with advisory and required enforcement
- trusted maintainer `approved` and `skip` decisions bound to the exact pull request head SHA
- a read-only `pull_request_target` gate contract that does not check out or execute contributor
  code

## Improved

- malformed merged-pull-request event rejection before collection
- GitHub API base URL validation before authenticated requests
- isolated Action provider resolution, output rendering, and bounded failure summaries
- deterministic scale and real CLI file-I/O regression guards

Existing dry-run, propose, commit, stage-draft, and promote-draft behavior remains compatible. npm
CLI package versions remain independent from Action release versions.
